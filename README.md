# Personal Portfolio Website

## Project Description

A responsive personal portfolio website developed to showcase personal information, skills, projects, and contact details in a clean and visually appealing manner.

The website serves as an online professional profile and demonstrates fundamental front-end web development skills using HTML and CSS.

## Features

- Responsive portfolio design
- Personal introduction section
- Skills and project showcase
- Image integration
- Simple and user-friendly interface
- Deployed online using Vercel

## Technologies Used

- HTML5
- CSS3
- Vercel Deployment
